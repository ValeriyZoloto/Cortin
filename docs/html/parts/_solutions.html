ПРОЕКТИРУЕМ РЕШЕНИЯ ДЛЯ НЕСТАНДАРТНЫХ ОКОН И ПРОЁМОВ

  • НА АРОЧНОЕ ОКНО

    Профильный карниз монтируют в арочную нишу. Благодаря гибкому алюминиевому материалу, карниз может принимать форму любого изгиба (полуциркульные, мавританские, готические, английские)

  • ЭРКЕРНЫЙ КАРНИЗ

    Профильный алюминиевый карниз, в отличие от прямого, производится из нескольких частей, которые соединены между собой в один карниз. Отличным решением для эркера станут многослойные варианты с сочетанием лёгких полотен
